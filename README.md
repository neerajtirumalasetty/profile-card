# Profile Card UI

## Overview
This project is a simple and responsive Profile Card built using only HTML and CSS. It displays basic personal information along with social media links, focusing on clean design, layout structure, and modern styling techniques without using JavaScript or external libraries.

## Technologies Used
- HTML5
- CSS3
